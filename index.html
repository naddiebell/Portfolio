Internship at Savely

Live Website!

Project Description

Savely's goal is to give young people control and knowledge of their personal finances.

During my internship, I was part of a team that built a working web application for Savely from scratch. I learned a lot about Agile Development as each week we would talk to the Savely team who gave us mockups and wireframes. We then asked questions and prioritized what the clients wanted. Our team used Trello to map the user's experience and organize tasks.

It was a great experience for me to get involved in seeing a real world application come to life from developing components to setting up the backend and seeing the application deployed.

Tools and Frameworks Used

  • JavaScript, Node, Express, React
  • Mongo DB
  • Google Firestore
  • Material UI
  • Nivo Charts
  • Restful API design
  • Testing with Jest, Mocha, & Chai

Stockholm's American Retro Movie Theater

Live Website!

Project Description

The concept is a Retro Movie Theater in Stockholm which plays American classic films. I created a fully working site, where a user can browse the website, look at movie trailers, buy an amount of tickets integrated with Stripe checkout and is then sent an email confirmation.

I wanted to test how I could maximize a User’s experience by utilizing features such as translation and local storage to remember the User’s choice even if they refresh the page, easy site navigation and fast payment with Stripe. It was fun to show off my React skills and to think about what is important from the User’s perspective.

Tools and Frameworks Used

  • React, JavaScript, Node, Express
  • Mongo DB
  • Stripe
  • I18n translation
  • Hosted on Netlify and Heroku
